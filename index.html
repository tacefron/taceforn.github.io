スキル

Linux コマンドライン

OSとLinuxの概要や操作方法を始め、Teratermを使用したリモートログイン方法を学び、基本コマンドとしては pwd,cd,ls,vi,mkdir,cp,mv,rm,cat,manを使用できます。実習としてはディレクトリとファイル作成、移動、複製をしておりました。

HTML、CSS作成

FREE CODE CAMP内で主に学習いたしました。VScodeを用いてコードの書き写しなどして学習しカラフルなページレイアウトや配置、拡大縮小などを用いたWEBページの作成をすることができます。

Javascript

基本的な変数や関数と条件の使用方法を学びJavaScriptカウンターアプリを作成しました。現在も学習を続けております。
